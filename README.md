# domoticz-ical-plugin
Domoticz iCal Plugin
--------------------


Installation
------------

In your `domoticz/plugins` directory do

```bash
git clone https://github.com/nd2014-public/domoticz-ical-plugin.git
```

Restart your Domoticz service with:

```bash
sudo service domoticz.sh restart
```
